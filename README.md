# Creative Code Collection

This repository contains a collection of creative coding projects developed using p5.js, HTML, CSS, and JavaScript. Each project explores different aspects of interactivity, animation, sound synthesis, and dynamic visuals.

## Projects

- **Animated Circle**: Interactive animation of shapes reacting to mouse movement.
- **Canvas Basics**: Introduction to HTML5 canvas drawing.
- **Canvas Sounds**: Visual sound interactions triggered by user input.
- **Control Canvas**: Manipulating canvas elements through user controls.
- **Sound Garden**: Audio-based interactive visual experience.
- **Sound Synthesis**: Experimenting with Web Audio API for generative sound design.
- **Perlin Noise Demos**: Procedural animation using Perlin noise algorithms.
- **Calculator**: Simple calculator built with HTML, CSS, and vanilla JavaScript.
- (More projects to be added...)

## Technologies Used

- p5.js
- JavaScript (Vanilla)
- HTML5
- CSS3

## How to Run

1. Open any project folder.
2. Open the `index.html` file inside that folder with a web browser.
3. Interact with the canvas to see animations, sounds, or effects.
